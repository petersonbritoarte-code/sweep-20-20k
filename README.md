# Frequência 20/20
**by Prof. Peterson Brito**

Ferramenta didática para estudo e demonstração auditiva:
- Sweep logarítmico (20 Hz → 20 kHz)
- Ruído branco
- Ruído rosa
- Interface por faixas de frequência (subgrave → super agudo)
- Funciona offline (PWA)

## Como usar
1. Abra o app no navegador.
2. Ajuste o volume (comece baixo).
3. Escolha o modo (Sweep / Branco / Rosa).
4. Clique em **Iniciar**.

## Segurança
Comece com volume baixo. Sweeps/ruídos podem ficar agressivos em 2–5 kHz.  
Evite fones no máximo, faça pausas e interrompa ao menor sinal de desconforto.

# Frequency 20/20
**by Prof. Peterson Brito**

A teaching tool for hearing and frequency-range demonstrations:
- Log sweep (20 Hz → 20 kHz)
- White noise
- Pink noise
- Frequency bands UI (sub-bass → ultra-high)
- Works offline (PWA)

## How to use
1. Open the app in your browser.
2. Set volume (start low).
3. Select mode (Sweep / White / Pink).
4. Press **Start**.

## Safety
Start with low volume. Sweeps/noise can be harsh around 2–5 kHz.  
Avoid max headphone volume, take breaks, stop if you feel discomfort.
